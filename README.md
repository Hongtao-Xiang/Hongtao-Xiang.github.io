# Hongtao-Xiang.github.io

个人主站，使用 GitHub Pages 发布。

## 站点结构

```text
/
  index.html
  articles/
  videos/
  resources/
    codex-handbook/
  about/
  assets/
```

## 更新方式

修改页面后提交并推送：

```bash
git add .
git commit -m "Update site"
git push
```

GitHub Pages 会自动从 `main` 分支发布到：

```text
https://hongtao-xiang.github.io/
```
